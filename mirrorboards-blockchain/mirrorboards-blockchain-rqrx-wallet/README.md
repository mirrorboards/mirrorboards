# README

## Quick Start

This repository provides a simple setup for running and interacting with the R-Squared Blockchain environment. Follow the steps below to get started.

---

### 1. One-Time Setup

Run the following command to start the required services:

```bash
docker-compose up -d
```

### 2. Start the Wallet

After starting the services, initialize the CLI wallet by running:

```bash
./wallet.sh
```

---

### Additional Resources

CLI Wallet
The CLI wallet is available at:

```
/usr/local/bin/cli_wallet
```
